# Usage
<!-- usage -->
```sh-session
$ npm install -g envo
$ envo COMMAND
running command...
$ envo (-v|--version|version)
envo/0.0.0 darwin-x64 node-v10.15.1
$ envo --help [COMMAND]
USAGE
  $ envo COMMAND
...
```
